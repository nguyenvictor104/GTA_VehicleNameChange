-VehicleNameChange-
Description:
A simple script that allows you to change an existing vehicle name to a new one.
Example:
You can easily rename a file from POLICE3.yft -> SHERIFF.yft

-DeleteVehicleFiles-
Description:
A script that deletes .xml, .ytd, and .ytf files once you're done renaming them.

Supports these files:
*.xml
*.ytd
*.ytf
*_hi.ytf
